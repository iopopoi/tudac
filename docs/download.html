다운로드

1. 쉘에서 다운로드

git clone git@github.com:iopopoi/tudac.git

위 명령어를 입력해서 다운로드 합니다.

github에 SSH키가 등록되지 않았다면, 읽기전용으로 다운로드 합니다

git clone https://github.com/iopopoi/tudac

읽기전용 다운로드는 위와 같습니다.




2. django 다운로드

*되도록 파이썬 3.8.0, 가상환경에서 진행해주세요


pip3 install django

위 명령어를 입력하면 django 다운로드가 완료됩니다.
다운로드가 실패한다면, python3이나 pip3을 재설치해보고, django를 재설치해주세요.


pip3 uninstall django

django 삭제는 위와 같습니다.
