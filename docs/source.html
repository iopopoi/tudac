소스코드 기여방법

1. 프로젝트를 Fork 합니다.
https://github.com/[YOUR ID]/tudac

그러면 위 링크, 즉 본인의 새로운 프로젝트로 생성될 것입니다.

git clone https://github.com/[YOUR ID]/tudac

2. Fork한 개인 저장소를 로컬에 클론합니다.

git checkout -b [BRANCH NAME]

3. 적절한 이름으로 토픽 브랜치를 생성합니다.

git commit -a -m "[COMMIT MESSAGE]

4. 소스코드를 수정하고, 토픽 브랜치에 커밋합니다.

git push origin [BRANCH NAME]

5. 깃허브의 개인 저장소에 토픽 브랜치를 푸쉬합니다.

6. 깃허브에 들어가서 pull request를 남깁니다.
